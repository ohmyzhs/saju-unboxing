      브라우저

        유입 페이지

          운영 체크리스트

          결제 게이트

          결제 승인 전에는 분석 생성이 열리지 않습니다. Toss 오류는 결제 시작 이벤트로 기록합니다.

          방문자 추적

          IP, 사용자 에이전트, 기기, 화면, UTM, 레퍼럴, 진입 페이지, 마지막 페이지를 이벤트로 쌓습니다.

          퍼널 관리

          상품 선택, 주문 확인, 결제 시작, 결제 오류, 분석 시작, 분석 완료를 단계별로 봅니다.

          상품 운영

          상품별 가격과 프롬프트를 분리해서 운영하고, 저장 즉시 사용자 화면에 반영합니다.

          방문자 분석

          서버가 받은 IP와 브라우저가 보낸 기기 정보를 함께 표시합니다. 실제 운영에서는 개인정보 처리방침에 수집 항목을 명시해야 합니다.

          최근 30명 방문 여정

          방문자별 페이지 이동·이벤트 흐름을 시간순으로 보여줍니다. 결제까지 도달한 동선과 이탈 지점을 함께 확인할 수 있어요.

          전환 퍼널

          유입/UTM 분석

          이탈 페이지

          브라우저 종료/이동 시점에 마지막으로 머물던 화면과 체류 시간을 기록합니다.

          상품·가격 관리

          가격을 0원으로 저장하면 사용자 화면에서 무료 상품으로 처리됩니다.

          이미지 관리

          메인 배너와 상품 썸네일을 교체합니다. 업로드 즉시 저장되고, 사용자 화면 새로고침 시 반영됩니다. (4MB 이하 권장)

          사업자 정보 (사이트 하단 표기 · 토스 카드사 심사 필수)

          토스에 제출한 사업자등록증과 100% 동일하게 입력하세요. 저장 후 사용자 화면 새로고침 시 하단 푸터에 반영됩니다.

          약관 · 개인정보 · 환불 규정

          비워두면 우리가 제공하는 기본 문구가 자동 노출됩니다. "기본문구 불러오기"로 채운 뒤 사업 형태에 맞게 검토·수정하세요. ⚠️ 법적 효력이 있는 문서입니다.

          최근 거래 내역

          분석 작업

          만세력 API 연결

          본사(대표)에게 받은 만세력 API 주소와 내 전용 키를 여기에 입력하세요. 여기에 입력하면 .env(환경변수)에 SAJU_API_BASE·SAJU_API_KEY를 따로 넣지 않아도 됩니다. (여기 값이 우선 적용되고, 비워두면 환경변수를 사용합니다. 응답 매핑은 자동이라 설정 불필요)

          AI 해설 모델

          사주 해설을 만드는 AI 모델입니다 (OpenCode Go). 서버가 모델별 API와 JSON 응답 형식을 자동 처리합니다. 비워두면 환경변수 OPENCODE_MODEL → 기본 glm-5.2를 사용합니다.

          챗봇 상담 모델

          강무영 상담각이 답변에 쓰는 모델입니다. 리포트 해설 모델과 별개로 설정합니다. 비우면 위 해설 모델을 따라갑니다. 빠른 응답엔 deepseek-v4-flash를 권장합니다.

          관리자 비밀번호 변경

          로그인에 쓰는 비밀번호를 여기서 바꿉니다. 변경 즉시 적용되고, 다음 로그인부터 새 비밀번호를 사용합니다.
