❌ 항목별 안내:
• 중앙 만세력 API — 대표에게 받은 SAJU_API_BASE·SAJU_API_KEY를 넣으세요. 포인트 잔액도 함께 표시됩니다.
• Supabase — SUPABASE_URL·SUPABASE_SERVICE_KEY 입력 + supabase/schema.sql을 SQL Editor에 1회 실행.
• OpenAI — OPENAI_API_KEY 입력(해설 생성용).
• 토스/카카오 — 없어도 학습은 됩니다(토스는 테스트키 자동). 고객 결제·로그인 쓰려면 키 입력.
